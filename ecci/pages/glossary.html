English Term Español Categoría Definition / Definición
Responsive Design Diseño Responsivo FrontEnd Design approach that makes web pages render well on a variety of devices and screen sizes by using flexible grids, layouts, and CSS media queries.
CSS Grid Cuadrícula CSS FrontEnd A two-dimensional CSS layout system that divides a page into rows and columns, enabling precise placement of elements without floats or positioning.
Flexbox Caja Flexible FrontEnd A one-dimensional CSS layout method for arranging items in a row or column, providing powerful alignment and distribution capabilities.
Media Query Consulta de Medios FrontEnd A CSS technique that applies different styles based on the characteristics of the user's device, such as screen width, resolution, or orientation.
DOM (Document Object Model) Modelo de Objeto del Documento FrontEnd A programming interface for web documents that represents the page as a tree of objects, allowing JavaScript to dynamically modify content and structure.
Semantic HTML HTML Semántico FrontEnd The use of HTML elements according to their intended purpose (e.g., <header>, <article>, <nav>), improving SEO, accessibility and code readability.
CSS Custom Properties Variables CSS FrontEnd CSS variables defined with -- prefix that store reusable values (colors, fonts, sizes), enabling consistent theming and easier maintenance across stylesheets.
LocalStorage Almacenamiento Local FrontEnd A Web API that allows JavaScript to store key-value pairs in the browser with no expiration date, persisting data across sessions without a server.
Event Listener Escuchador de Eventos FrontEnd A JavaScript procedure that waits for a specific event (click, scroll, input) on a DOM element and executes a callback function when the event occurs.
Intersection Observer Observador de Intersección FrontEnd A JavaScript API that asynchronously observes changes in the intersection of a target element with the viewport, used for lazy loading and scroll animations.
User Experience (UX) Experiencia de Usuario UX/UI The overall experience a person has when interacting with a product or service, encompassing usability, accessibility, performance, and emotional response.
User Interface (UI) Interfaz de Usuario UX/UI The visual and interactive elements through which a user interacts with a digital product: buttons, menus, forms, typography, icons, and layout.
Wireframe Esquema de Pantalla UX/UI A low-fidelity visual guide representing the skeletal structure of a digital interface, focusing on layout and functionality without visual design details.
Usability Usabilidad UX/UI The ease with which users can learn and use a product to achieve their goals effectively, efficiently, and with satisfaction.
Information Architecture Arquitectura de Información UX/UI The structural design of shared information environments, organizing and labeling content in an effective and sustainable way to support usability and findability.
Microinteraction Microinteracción UX/UI Small, contained product moments that accomplish a single task — like a button hover animation or a form validation indicator — that improve overall UX.
Affordance Affordance / Señal de Uso UX/UI A design property that communicates how an object should be used — a button that looks pressable, a link that appears clickable — guiding user interaction intuitively.
Color Theory Teoría del Color Color Theory A body of practical guidance for color mixing and the visual effects of specific color combinations, based on scientific and artistic principles.
Pantone Matching System Sistema de Igualación Pantone Color Theory A standardized color reproduction system used globally in design, printing, and manufacturing to ensure consistent color communication across media.
Semiology of Color Semiología del Color Color Theory The study of color as a sign system, analyzing the cultural, psychological, and contextual meanings that colors communicate beyond their physical properties.
Color Wheel Círculo Cromático Color Theory A circular diagram organizing colors by their chromatic relationships, first systematized by Isaac Newton (1704), used to identify harmonious color combinations.
Complementary Colors Colores Complementarios Color Theory Colors located opposite each other on the color wheel that, when placed together, create high contrast and visual vibrancy; used strategically in UI design.
Color Psychology Psicología del Color Color Theory The study of how colors affect human perception, emotion, and behavior, applied in marketing, branding, and digital product design to guide user decisions.
Machine Learning Aprendizaje Automático AI / IA A subset of AI enabling systems to learn from data and improve performance on tasks without being explicitly programmed, through statistical pattern recognition.
Generative AI IA Generativa AI / IA AI systems capable of generating new content — text, images, code, audio — based on patterns learned from training data. Examples: GPT-4, Claude, DALL-E.
Algorithmic Bias Sesgo Algorítmico AI / IA Systematic and unfair discrimination in AI outputs caused by biased training data, leading to inequitable outcomes across demographic groups.
Natural Language Processing Procesamiento de Lenguaje Natural AI / IA An AI field focused on enabling computers to understand, interpret, and generate human language — powering chatbots, translation, and voice assistants.
Grid System Sistema de Cuadrícula Bootstrap Bootstrap's 12-column responsive layout system based on Flexbox, enabling developers to build complex layouts with breakpoint-specific column sizing.
Breakpoint Punto de Quiebre Bootstrap Predefined screen width thresholds in Bootstrap (xs, sm, md, lg, xl, xxl) that trigger different layout behaviors for responsive design adaptation.
Utility Classes Clases de Utilidad Bootstrap Single-purpose CSS classes in Bootstrap (e.g., .mt-3, .d-flex, .text-center) that apply a specific style, reducing the need for custom CSS.
Component Componente Bootstrap A reusable UI building block in Bootstrap — such as a Card, Modal, Navbar, Carousel, or Accordion — combining HTML, CSS, and JavaScript behavior.
Version Control Control de Versiones Software Eng. A system (e.g., Git) that records changes to files over time, enabling teams to track history, revert changes, and collaborate on code without conflicts.
API (Application Programming Interface) Interfaz de Programación de Aplicaciones Software Eng. A set of protocols and definitions that allows different software systems to communicate, enabling developers to use external functionality without knowing its implementation.
Agile Methodology Metodología Ágil Software Eng. An iterative approach to software development that delivers value in small, usable increments through collaboration, customer feedback and adaptation to change.
DevOps Desarrollo y Operaciones Software Eng. A set of practices combining software development and IT operations to shorten the development lifecycle and provide continuous delivery of high-quality software.
Clean Code Código Limpio Software Eng. Code that is easy to read, understand, and maintain — characterized by meaningful names, small functions, clear comments, and no duplication (DRY principle).
WCAG (Web Content Accessibility Guidelines) Pautas de Accesibilidad Web Accesibilidad International standards published by W3C defining how to make web content more accessible to people with disabilities, organized around four principles: Perceivable, Operable, Understandable, Robust.
ARIA (Accessible Rich Internet Applications) Aplicaciones de Internet Enriquecidas y Accesibles Accesibilidad A set of HTML attributes that define ways to make web content and applications more accessible to assistive technologies like screen readers.
Color Contrast Ratio Relación de Contraste de Color Accesibilidad A measure of the luminance difference between foreground text and background color. WCAG AA requires a minimum ratio of 4.5:1 for normal text to ensure readability.
